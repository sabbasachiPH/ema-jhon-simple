# Ema-jon simple <h1>
**This project is a full featured e-commerce site**

*Main Features:*
1. Select Item From the available items 
1. See Item details 
1. Order Review 
1. LogIn with Google account or email 
1. Payment using stripe

## Technoloy used in this project <h2>
  1. React
  1. Express.js
  1. Node.js
  1. MongoDB
  1. Firebase
  1. Stripe
  1. Heroku
